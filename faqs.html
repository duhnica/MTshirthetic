What is the difference between OVERSIZED size tag and ANY SIZE size tag?

  OVERSIZED size tag is just one size which is oversize but if you cant to change the size into REGULAR SIZE such as small, medium, large, or extra large, you need to pay a 20.00 (additional fee). While ANY SIZE size tag is available for small, medium, large, extra large, and oversize.*no additional fee.

Example for Oversized size tag:
Oversize = 288.00
Oversize + Small(regular size)= 288.00 + 20.00 = 308.00 and so on..
